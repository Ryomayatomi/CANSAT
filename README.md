# CANSAT Project

## Overview

This repository contains the development records and source code for a CanSat project built by a student of Toyota National College of Technology.

A CanSat (Can Satellite) is a small satellite-like system packed into a can-sized container. The objective of this project is to acquire practical experience in embedded systems, sensor integration, telemetry, and autonomous control.

---

## Objectives

* Learn embedded system development using ESP32
* Acquire and process sensor data
* Record environmental data during operation
* Develop autonomous recovery and navigation algorithms
* Apply engineering design processes used in aerospace projects
